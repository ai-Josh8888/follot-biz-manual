# 通知管理

Follot Bizからの通知設定や、システムからのお知らせを確認する方法を説明します。

## 通知設定

個人のアカウントに対する通知の受け取り方を設定できます。

-   **メール通知**: 特定のアクション（例：タスクの割り当て、承認依頼など）があった場合にメールで通知を受け取るかどうかを設定します。
-   **デスクトップ通知**: ブラウザを開いているときに、リアルタイムでデスクトップに通知を表示するかどうかを設定します。

## お知らせの確認

システムメンテナンスや新機能のリリースなど、運営からのお知らせを確認できます。通常、画面上部のベルアイコンなどから確認できます。
