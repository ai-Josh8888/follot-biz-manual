# 基本操作：ログインと初期設定

アプリケーションを利用するための最初のステップです。

## 2-1. ログイン画面へのアクセス

1.  ウェブブラウザを開き、指定されたURL（ `https://follot-biz.web.app/login` ）にアクセスします。
2.  ログイン画面が表示されます。

    `[画像：ログイン画面]`

## 2-2. ログイン方法

1.  ご自身の**メールアドレス**と**パスワード**をそれぞれ入力します。
2.  **「ログイン」**ボタンをクリックします。

## 2-3. 組織とテナントの選択

ログイン後、利用する組織やテナントを選択する画面が表示される場合があります。

> **Note**
> 所属する組織やテナントが1つしかない場合は、この画面は表示されずにホーム画面へ直接移動します。

1.  **「組織を選択してください」**という画面が表示されたら、所属する組織を選択します。
2.  次に**「テナントを選択」**画面が表示されたら、プルダウンメニューから利用するテナント（例：「クーバル株式会社」）を選択します。
3.  **「決定」**ボタンをクリックすると、ホーム画面（トップページ）に移動します。
