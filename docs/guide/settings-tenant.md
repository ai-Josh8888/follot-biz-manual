# テナント・組織設定

**※この設定は、管理者権限を持つユーザーのみ表示・編集が可能です。**

テナント（組織）全体の情報や、アプリケーションの基本的な動作を設定します。

## 主な設定項目

*   **契約情報**: 組織名やインボイス情報などを管理します。
*   **アプリ設定**:
    *   **一般設定**: アプリケーション全体の共通設定を行います。
    *   **機能の有効化**: Teamチャット、Bizトーク、AI機能など、テナント内で利用する機能を個別にON/OFFできます。OFFにした機能は、ユーザーの画面に表示されなくなります。
*   **外部サービス連携**: LINEやChatWorkなど、外部サービスとの連携設定を行います。（詳細は各連携ページを参照）
*   **ブラックリスト管理**: 特定のメールアドレスからの受信を拒否するブラックリストを設定します。
