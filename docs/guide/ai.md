# AI機能

AIを活用して、顧客対応やメッセージ作成を効率化する機能です。

## プロンプト管理

よく使う指示文（プロンプト）をテンプレートとして保存・管理できます。

-   **テンプレート作成**: 「〇〇についてのメール文を作成して」といったプロンプトを、変数（質問）付きで作成できます。
-   **共有範囲設定**: 作成したプロンプトを「自分だけ」で使うか、「チーム全員」で共有するかを設定できます。

`[画像：プロンプト管理画面]`

## AIチャット

作成したプロンプトを使ったり、自由に質問したりしてAIと対話できるチャット機能です。

-   **BizトークやWorkチャットとの連携**: チャットの入力フォームから直接AI（プロンプト）を呼び出し、生成された文章をメッセージとして挿入できます。
