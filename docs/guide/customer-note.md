# 顧客ノート

顧客に関する記録をチャット形式で残すためのノート機能です。
顧客ごとに専用のチャンネル（スレッド）が作成され、社内での情報共有や引き継ぎに役立ちます。

## 主な機能

*   **ノートの投稿**:
    *   サイドバーから顧客ノートを選択すると、その顧客専用のチャット画面が表示されます。
    *   メッセージを入力して送信すると、タイムライン形式で記録が残ります。
    *   投稿したメッセージは後から編集・削除が可能です。

*   **ファイル管理**:
    *   ノートには画像やドキュメントなどのファイルを添付できます。
    *   サイドバーの「ファイル」メニューからは、自分が登録したファイルや、すべてのファイルを一覧で確認・管理することができます。
