# LINE連携設定

**※この設定は、管理者権限を持つユーザーのみ表示・編集が可能です。**

LINE公式アカウントとFollot Bizを連携させるための設定です。連携することで、顧客とのLINEでのやり取りをBizトーク内で一元管理できるようになります。

## 設定の主な流れ

1.  **LINE Developersコンソールでの準備**:
    *   LINE公式アカウント用のプロバイダーとチャネルを作成します。
    *   Messaging APIを有効にし、「チャネルアクセストークン」と「チャネルシークレット」を取得します。
2.  **Follot Bizでの設定**:
    *   「テナント設定」>「連携設定」>「LINE」を開きます。
    *   LINE側で取得した「チャネルアクセストークン」と「チャネルシークレット」を入力し、接続テストと保存を行います。
3.  **Webhook URLの設定**:
    *   Follot Bizで表示される「Webhook URL」を、LINE Developersコンソールのチャネル設定に貼り付けます。これにより、LINEでのイベント（メッセージ受信など）がFollot Bizに通知されるようになります。

## 友だち追加と承認

顧客がLINEで公式アカウントを友だち追加すると、Follot Bizの「LINE管理画面」に通知が届きます。管理者がこれを承認し、既存の顧客情報と紐付けることで、正式に連携が完了します。
