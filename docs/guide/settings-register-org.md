# 新規組織の登録

Follot Bizの利用を新たに開始する際に、組織（カンパニー）の情報を登録するための機能です。

## 招待された場合

管理者から招待メールを受け取った場合、メール内のリンクをクリックすると招待承認ページに移動します。

*   **初めて利用する場合**:
    アカウント作成画面が表示されます。画面の指示に従い、名前やパスワードを設定してアカウントを作成してください。

*   **既にアカウントをお持ちの場合**:
    ログインすることで、新しい組織への参加が承認されます。

## 管理者が新規に組織を作成する場合

**※この操作は、特別な権限を持つ管理者のみ実行可能です。**

1.  `/register-organization` のURLにアクセスします。
2.  「オーガニゼーションの作成」フォームが表示されます。
3.  会社名や住所などの必須項目を入力し、「送信」ボタンをクリックします。
4.  処理が成功すると、組織が作成され、完了メッセージが表示されます。
