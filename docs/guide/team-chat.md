# Teamチャット (社内チャット)

社内メンバー間のリアルタイムコミュニケーションを目的としたチャット機能です。

## 主な機能

*   **ルームとDM**: 複数人で会話する「ルーム」と、1対1で会話する「ダイレクトメッセージ(DM)」が利用できます。
*   **メッセージの編集・取消**: 自分が送信したメッセージは、後から編集したり、取り消したりすることができます。
*   **スレッド**: 特定のメッセージに対して返信を重ねることで、会話の流れを整理できます。
*   **メンション**: `@ユーザー名` を付けてメッセージを送信すると、相手に通知を送ることができます。
*   **ファイル共有**: 画像やドキュメントなどのファイルを共有できます。
*   **絵文字リアクション**: 各メッセージに絵文字でリアクションを付けることができます。
