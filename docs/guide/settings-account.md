# アカウント設定

ご自身のユーザー情報を設定・変更する画面です。

## 主な設定項目

*   **アカウント名・表示名**: アプリケーション内で表示される名前を変更できます。
*   **メールアドレス**: ログインに使用するメールアドレスを変更できます。変更後、確認メールが送信されます。
*   **パスワード変更**: ログインパスワードを変更できます。セキュリティのため、定期的な変更をおすすめします。
*   **アイコン設定**: プロフィールアイコンとして表示される画像を設定できます。

## アクセス方法

1.  画面左下の歯車アイコンなどから「設定」画面に移動します。
2.  設定画面のサイドバーから「アカウント」を選択します。
