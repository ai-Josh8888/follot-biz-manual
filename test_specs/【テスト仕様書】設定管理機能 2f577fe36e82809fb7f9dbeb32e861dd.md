## 5. 設定管理機能 (/settings)

### 基本情報

- **URL**: `/settings`
- **コンポーネント**: `src/components/environments/Settings/`
- **機能概要**: ユーザー個人のアカウント情報、組織の基本情報、およびアプリケーション全体の機能（チャット、AI 等）の有効/無効設定を管理する。

### テスト項目

### 1. アカウント設定 (ACCOUNT)

| 項目 ID | 確認内容 | 期待結果 | 実施者 | 実施日 | 結果 | 備考 |
| --- | --- | --- | --- | --- | --- | --- |
| SET-ACC-001 | ページアクセス | サイドバー「アカウント」をクリック。<br>【期待動作】<br>・アカウント名、メールアドレス、表示名、パスワード変更フォームが表示されること。 |  |  |  |  |
| SET-ACC-002 | 基本情報更新 | アカウント名や表示名を変更して「Submit」をクリック。<br>【期待動作】<br>・「更新に成功しました」と表示され、変更が保存されること。 |  |  |  |  |
| SET-ACC-003 | パスワード変更 | 新しいパスワード（8 文字以上、英数記号必須）と確認用パスワードを入力して保存。<br>【期待動作】<br>・更新が成功すること。<br>・不一致の場合はエラーが表示されること。 |  |  |  |  |
| SET-ACC-004 | メールアドレス変更 | メールアドレスを変更して保存。<br>【期待動作】<br>・「更新に成功しました。認証メールを送信しました」と表示されること。 |  |  |  |  |
| SET-ACC-005 | アイコン設定 | 画像ファイルを選択して保存。<br>【期待動作】<br>・ヘッダー等のアイコン表示が更新されること。 |  |  |  |  |

### 2. 組織設定 (ORG)

※ 権限を持つユーザー（管理者等）のみ実施

| 項目 ID | 確認内容 | 期待結果 | 実施者 | 実施日 | 結果 | 備考 |
| --- | --- | --- | --- | --- | --- | --- |
| SET-ORG-001 | ページアクセス | サイドバー「契約情報」配下の「契約情報」をクリック。<br>【期待動作】<br>・組織名やインボイス情報等のフォームが表示されること。 |  |  |  |  |
| SET-ORG-002 | 基本情報更新 | 組織名を変更して保存。<br>【期待動作】<br>・変更が反映されること。 |  |  |  |  |

### 3. アプリ設定 (FUNC)

※ 権限を持つユーザーのみ実施

| 項目 ID | 確認内容 | 期待結果 | 実施者 | 実施日 | 結果 | 備考 |
| --- | --- | --- | --- | --- | --- | --- |
| SET-FUNC-001 | 一般設定 | サイドバー「アプリ設定」＞「一般」をクリック。<br>【期待動作】<br>・アプリケーション全体の共通設定項目が表示されること。 |  |  |  |  |
| SET-FUNC-002 | 機能トグル | 各種機能（Team チャット、Biz トーク、AI チャット等）の ON/OFF 切り替え。<br>【期待動作】<br>・設定変更後、保存できること。<br>・OFF にした機能がメインサイドバー（画面左端）から非表示になること。 |  |  |  |  |

### 4. 権限・表示制御テスト (AUTH)

| 項目 ID | 確認内容 | 期待結果 | 実施者 | 実施日 | 結果 | 備考 |
| --- | --- | --- | --- | --- | --- | --- |
| SET-AUTH-001 | 一般ユーザー表示 | 組織設定やアプリ設定の権限を持たないユーザーでログイン。<br>【期待動作】<br>・設定サイドバーに「契約情報」「アプリ設定」「テナント設定」等の管理者向け項目が表示されないこと。<br>・「アカウント情報」のみ（または許可された範囲のみ）表示されること。 |  |  |  |  |
