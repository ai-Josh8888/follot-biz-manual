# チャット・トーク (Bizトーク)

Bizトークは、顧客とのコミュニケーションを一元管理するためのチャット機能です。
外部の顧客から送信されたメールは自動的に取り込まれ、顧客ごとに専用のチャットルームが作成されます。

## 機能概要

-   **自動ルーム作成**: 顧客からの初回メール受信時に、顧客情報と専用チャットルームが自動で生成されます。
-   **一元管理**: メール、LINE、Chatworkなど、異なるツールからの連絡もBizトーク内でまとめて確認・返信できます。
-   **社内での返信**: 社内の担当者は、この画面から直接顧客へメッセージを送信できます。
-   **承認フロー**: ユーザーの権限によっては、メッセージ送信に上長などの承認が必要になる場合があります。
