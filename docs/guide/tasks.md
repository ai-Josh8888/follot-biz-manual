# タスク管理

タスクのライフサイクル管理（作成、割当、進捗管理、完了）を行う機能です。サイドバーによるフィルタリングが特徴です。

## 1. タスク一覧画面の見方
サイドメニューの「タスク管理」をクリックすると、タスク一覧画面が表示されます。

### サイドバーフィルタ
サイドバーの項目をクリックすることで、表示するタスクを絞り込めます。
*   **全タスク**: すべてのタスク
*   **作成タスク**: 自分が作成したタスク
*   **担当タスク**: 自分に割り当てられたタスク
*   **完了タスク**: ステータスが「完了」のタスク
*   **関連ルームタスク**: グループチャット（BizトークやTeamチャット）で作成されたタスク

### ページネーション（リストの読み込み）
*   **通常時:** 最新の20件が表示され、リスト下部の「続きを読む」ボタンで次の20件を読み込みます。
*   **検索時:** キーワード検索を実行すると、条件に一致するタスクがすべて読み込まれ、クライアント側でページ分割されます。

## 2. 新規タスクの作成

1.  サイドバー上部の「タスク新規作成」ボタンをクリックします。
2.  タスク情報を入力するためのポップアップ画面が表示されます。
3.  必須項目（タイトル）や、担当者、期限、内容などを入力します。
4.  **「作成」** ボタンをクリックすると保存され、一覧に追加されます。

## 3. タスクの編集・削除

1.  一覧から編集したいタスクをクリックすると、編集用のポップアップ画面が表示されます。
2.  内容を修正し、**「更新」**ボタンをクリックして保存します。
3.  タスクを完了にするには、ステータスを「完了」に変更して保存します。
4.  タスクを削除するには、編集画面内の**「削除」**ボタンをクリックします。
